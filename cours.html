🛠️ Développement de l'application IA de suggestion de recettes 🍳🤖

1️⃣On commence par poser l’idée 💡
On va sur tldraw.com pour dessiner l’architecture de notre application.
2️⃣On génère le frontend avec Lovable 🎨
On utilise Lovable AI pour créer une interface simple et propre.
3️⃣On exporte le frontend sur GitHub ☁️
On connecte Lovable à notre repo GitHub et on envoie le code.
4️⃣On récupère le projet en local 💻
On clone le repo :
git clone https://github.com/mon-compte/mon-projet.git
5️⃣On installe Cursor AI 🧠
On utilise Cursor pour un développement assisté par IA.
6️⃣On active notre environnement Python 🐍
python -m venv env
source env/bin/activate
7️⃣On ajoute le backend avec FastAPI 🚀
pip install fastapi uvicorn python-dotenv
8️⃣On configure notre clé API OpenAI 🔐
On ajoute OPENAI_API_KEY=sk-... dans un fichier .env
9️⃣On teste tout ! 🧪
uvicorn main:app --reload
On vérifie que les recettes s’affichent 🍝
🔟On pousse le tout sur GitHub 🚀
git add .
git commit -m "Ajout backend + intégration OpenAI"
git push
